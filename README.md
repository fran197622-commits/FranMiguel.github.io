Datos sobre los archivos:
El proyecto de la tienda de zapatos se ubica en el apartado principal (o raíz), que sería el inicio. Además, está dividido en otros tres archivos,
,que vienen siendo los contactos, el apartado "nosotros" y la tienda.
Se destaca un intento de uso de un script en la parte de "index" (raíz del html), pero debido a la falta de tiempo, este no funciona correctamente, y no se muestran las demás imágenes

al igual que se hace con la ejecución del comando "flex", que interactúa con el sitio con otros comandos, que cambian el orden de la página sin necesidad de retocar el html.
No obstante, se aclara que, se puso varios cambios de tamaño en el propio html de cada archivo.html debido a la falta de tiempo y que no se me ocurrían otras ideas en aquel momento.
